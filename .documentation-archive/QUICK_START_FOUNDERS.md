## Troubleshooting

### Can't Log In
1. Check email is correct
2. Verify password is correct
3. Try clearing browser cache
4. Use incognito/private mode
5. Contact tech support

### Password Change Fails
1. Ensure current password is correct
2. New password must be 6+ characters
3. Confirm passwords match
4. Check internet connection
5. Try refreshing the page

### Email Not Working
1. Verify email account exists
2. Check credentials are correct
3. Test IMAP/SMTP connection
4. Check mail server status
5. Contact system admin
