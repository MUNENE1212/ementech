## Common SEO Issues to Avoid

❌ Don't ignore mobile optimization
❌ Don't duplicate content across pages
❌ Don't buy low-quality backlinks
❌ Don't keyword stuff (natural is better)
❌ Don't forget to update sitemap when adding pages
❌ Don't ignore page speed
❌ Don't skip alt tags on images

## Emergency SEO Checklist

If traffic drops suddenly:
1. Check Google Search Console for manual penalties
2. Verify site is accessible (not down)
3. Check for recent algorithm updates
4. Review recent changes to site
5. Check for hacked content
6. Verify robots.txt didn't block important pages
7. Test structured data still valid
