# Admin Dashboard - UI Wireframes & Layouts

## Overview

This document describes the UI layouts, wireframes, and interaction patterns for the EmenTech Admin Dashboard. These wireframes serve as blueprints for the implementation team.

---

## Design System

### Color Palette

**Primary Colors**:
- Primary: `#3B82F6` (Blue)
- Success: `#10B981` (Green)
- Warning: `#F59E0B` (Amber)
- Error: `#EF4444` (Red)
- Info: `#6366F1` (Indigo)

**Neutral Colors**:
- Background: `#FFFFFF` (Light mode), `#0F172A` (Dark mode)
- Surface: `#F8FAFC` (Light mode), `#1E293B` (Dark mode)
- Border: `#E2E8F0` (Light mode), `#334155` (Dark mode)
- Text Primary: `#0F172A` (Light mode), `#F8FAFC` (Dark mode)
- Text Secondary: `#64748B` (Light mode), `#94A3B8` (Dark mode)

**Status Colors**:
- Up: `#10B981` (Green)
- Down: `#EF4444` (Red)
- Degraded: `#F59E0B` (Amber)
- Unknown: `#94A3B8` (Gray)

### Typography

**Font**: Inter (system font stack)

**Headings**:
- H1: 36px, Bold, Line height 1.2
- H2: 30px, Bold, Line height 1.3
- H3: 24px, Semibold, Line height 1.4
- H4: 20px, Semibold, Line height 1.5

**Body**:
- Large: 18px, Regular
- Base: 16px, Regular
- Small: 14px, Regular
- XSmall: 12px, Regular

### Spacing Scale

- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px

### Components

All components from **shadcn/ui** with custom theming:
- Buttons (Primary, Secondary, Ghost, Danger)
- Cards
- Inputs, Textareas, Selects
- Modals/Dialogs
- Dropdowns
- Tables
- Tabs
- Badges
- Alerts/Notifications
- Charts (Recharts)
